# Match Users Chat

This project is a Node.js application that allows users with the same interests to be matched and chat with each other.

## Installation

1. Install dependencies:

   ```sh
   npm install
   ```

2. Start the server:

   ```sh
   npm start
   ```

## Endpoints

- `POST /chat/send` - Send a message in a chat room.

## Dependencies

- Express
- Axios
- Body-parser
